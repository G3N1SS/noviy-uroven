# Новый уровень

Веб-игра (PWA) для оператора **T2** — бесконечный вертикальный климбер в стиле Doodle Jump.
Персонаж-«сигнал» прыгает по вышкам связи вверх через эпохи сетей 2G → 6G.

> «Другие правила. Новый уровень».

Полный контекст проекта — в [`docs/t2_novy_uroven_master_konspekt.md`](docs/t2_novy_uroven_master_konspekt.md).
Рабочие правила для Claude Code — в [`CLAUDE.md`](CLAUDE.md).

**Прод:** [novi-uroven.vercel.app](https://novi-uroven.vercel.app) (Vercel, автодеплой на push в `main`).

## Стек

React 18 + TypeScript 5 (strict) + Vite 5 · PixiJS 8 (WebGL) · Zustand · Vitest · PWA (Workbox) · IndexedDB

## Команды

```bash
npm install       # установка зависимостей
npm run dev       # dev-сервер (http://localhost:5173, доступен по сети — тюнинг на телефоне)
npm run build     # прод-сборка (tsc --noEmit + vite build)
npm run preview   # предпросмотр прод-сборки
npm run lint      # ESLint
npm run format    # Prettier
npm test          # Vitest (валидатор проходимости генерации)
```

## Работа с 3 устройств

Сел за устройство — `git pull`. Встал — `git add . && git commit && git push`.
Не оставлять несинхронизированным.

## Статус

**Этап 0 (Фундамент) — ✅** Vite+React+TS(strict), PixiJS-рендер, Vercel-автодеплой.

**Этап 1 (Ядро прыжка) — 🟡** Гравитация, one-way платформы, автопрыжок, wrap, камера-вверх, генерация с pooling, счётчик высоты. 3 схемы управления (наклон/свайп/зоны) с выбором на паузе, localStorage. Осталось: **цикл тюнинга физики на реальном устройстве** (DoD).

**Этап 2 (Геймплейная глубина) — 🟡 почти закрыт**

- 3 типа платформ (ВОЛС / РРЛ с осколками / движущаяся с шевронами) + landing-bounce.
- 5 эпох по высоте с кроссфейдом фона и баннерами.
- Кристаллы (+1 цепочками, +5 крупные) — **ложатся на реальные дуги прыжка** между опорами (arc через vx=const), гарантированно собираемы одним прыжком.
- Препятствия: помеха (глитч-облако с отбросом+блоком) и фейк-платформа с обязательной декой-парой. **Помехи гейтятся валидатором проходимости** (физика прыжка + анти-overlap с платформой) — облако не встанет там, где перекроет единственный путь.
- 3 бустера: Гигабэк (×2 кристаллы), MiXX-щит (заряды спасения-батут), SafeWall (иммунитет + подсветка фейков).
- Экран Game Over с высотой (стенсил), нарративом эпохи, рекордом и кошельком. **Рекорд и crystalTotal в localStorage** — переживают перезагрузку.
- Валидатор проходимости (Vitest, 10/10): опоры на каждом уровне, зазоры ≤ прыжка, гарантия ВОЛС, жадный climb, проходимость помех.
- Осталось: паттерн-генерация (5 базовых, опционально) + плейтест-цикл (обязательно по DoD).

**Этапы 3–9** (визуал T2, звук, IndexedDB-офлайн, бэкенд-синк, интеграции T2, полировка) — впереди по зависимостям.

Все числа игры — только в [`src/game/config/balance.json`](src/game/config/balance.json). Полный дневник по инкрементам — в [`CLAUDE.md`](CLAUDE.md).
